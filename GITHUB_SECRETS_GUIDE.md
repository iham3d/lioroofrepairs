### 5. PLESK_HOST
**You need to get this from your Plesk hosting provider.**

This is your server hostname or IP address.

**Where to find it:**
- Check your Plesk welcome email
- Or look at your Plesk login URL (e.g., `server123.hostingprovider.com`)
- Or contact your hosting provider

**Example values:**
```
server.lioroofrepairs.co.uk
```
or
```
185.123.45.67
```

---

### 6. PLESK_USER
**You need to get this from your Plesk hosting provider.**

This is your SSH username.

**Where to find it:**
- Usually the same as your Plesk username
- Or check: Plesk ‚Üí Websites & Domains ‚Üí lioroofrepairs.co.uk ‚Üí Web Hosting Access
- Or contact your hosting provider

**Example:**
```
lioroofrepairs
```

---

### 7. PLESK_WEB_ROOT
**You need to get this from your Plesk hosting provider.**

This is the absolute path to your website's directory on the server.

**Common Plesk format:**
```
/var/www/vhosts/lioroofrepairs.co.uk/httpdocs
```

**To find it:**
- SSH into your server and run `pwd` in your web directory
- Or check Plesk ‚Üí File Manager
- Or contact your hosting provider

### Step 2: Contact Your Hosting Provider

You need to get these 3 values from your hosting provider:
1. **PLESK_HOST** - Server hostname/IP
2. **PLESK_USER** - SSH username
3. **PLESK_WEB_ROOT** - Web directory path

**Email template you can send:**

```
Subject: SSH Access Details for lioroofrepairs.co.uk

Hi,

I need the following information for my domain lioroofrepairs.co.uk:

1. SSH hostname or IP address
2. SSH username
3. Absolute path to web root directory (httpdocs folder)

I'm setting up automated deployment via GitHub Actions.

Thank you!
```

### Step 3: Add All Secrets to GitHub

Once you have all 7 values, add them to GitHub:

1. Go to: https://github.com/iham3d/lioroofrepairs/settings/secrets/actions
2. Click **"New repository secret"** for each one
3. Add all 7 secrets listed above

### Step 4: Test Deployment

After all secrets are added:

1. Make a small change to trigger deployment:
   ```bash
   echo "# Deployment test" >> README.md
   git add README.md
   git commit -m "Test deployment"
   git push origin main
   ```

2. Watch the deployment:
   - Go to: https://github.com/iham3d/lioroofrepairs/actions
   - Click on the latest workflow run
   - Monitor the deployment progress
